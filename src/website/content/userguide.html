Userguide

PLEASE NOTE! The userguide has many links to Gradle's javadoc (to obtain detail information). Gradle is written in Groovy, therefore we need to use the GroovyDoc tool to generate the api documentation. GroovyDoc is work in progress. For example properties are not processed yet. Right now you might have to directly look into the source code to get the information you want. You might use Fisheye for this. The source code is also part of the Gradle distribution. - Apologies for the inconvenience PLEASE NOTE! The layout of the pdf version is much nicer compared to the html version.
