About Gradle

Gradle is a build system which uses Groovy instead of XML to write your build scripts.

Gradle gives you
  • a very flexible general purpose build tool like Ant.
  • if wanted, build-by-convention frameworks a la Maven (for example for Java projects). But we never lock you in!
  • a very powerful support for multi-project builds.
  • a very powerful dependency management (based on Ivy).
  • supports fully your existing Maven or Ivy repository infrastructure.
  • makes usings Ant tasks a bliss.
