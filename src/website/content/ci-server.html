Continous Integration

Gradle is continously build at the following locations:

JetBrains Teamcity

  • Windows Build
  • Linux Build
  • Nightly Build

Codehaus Bamboo

  • Linux Build

As we had many platform specific issues it is very important for us to have a continous build for multiple platforms.

None of those CI servers has Gradle or Groovy installed. Yet Gradle is build by them with Gradle with the help of the Gradle Wrapper.
